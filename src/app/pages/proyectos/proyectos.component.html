ID FASHION

El objetivo del proyecto es la creación de una aplicación web para la gestión completa de un sitio orientado a la compra-venta de productos de moda. Se debecentrar el desarrollo tanto en la parte del usuario que se acerca al contenido para realizar sus compras, como en la parte del gestor que se encarga de definir y maquetar todo el contenido de los productos a mostrar.
